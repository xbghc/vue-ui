# 🚀 快速开始

欢迎使用 @xbghc/vue-ui！本指南将帮助您在几分钟内开始使用我们的 Vue 3 组件库。

## 🎯 第一个示例

### 基础使用

```vue
<template>
  <div>
    <ToolTip content="Hello World!">
      <template #trigger>
        <button>悬停查看提示</button>
      </template>
    </ToolTip>
  </div>
</template>

<script setup lang="ts">
import { ToolTip } from '@xbghc/vue-ui';
</script>
```

### 高级用法

```vue
<template>
  <div>
    <ToolTip
      content="这是一个自定义样式的提示框"
      placement="top"
      :show-arrow="true"
      :hover-delay="500"
      :offset="10"
    >
      <template #trigger>
        <button>高级提示</button>
      </template>
    </ToolTip>
  </div>
</template>

<script setup lang="ts">
import { ToolTip } from '@xbghc/vue-ui';
</script>
```

## 🧩 核心特性

### 📦 Tree Shaking

支持按需导入，只打包您使用的组件：

```typescript
import { ToolTip } from '@xbghc/vue-ui';
// 只有 ToolTip 相关代码会被打包
```

### 🔒 TypeScript 支持

完整的类型定义，提供最佳开发体验：

```typescript
import type { ToolTipProps, ToolTipPlacement } from '@xbghc/vue-ui';

// 完整的类型提示和检查
const placement: ToolTipPlacement = 'top';
```

### 🎨 主题定制

通过 CSS 变量轻松定制组件样式：

```css
:root {
  --tooltip-bg-color: #333;
  --tooltip-text-color: #fff;
  --tooltip-border-radius: 6px;
  --tooltip-arrow-size: 6px;
}
```

## 🎯 开发哲学

### Modern First, No Legacy Support

本项目优先使用现代 Web 标准，不维护过时版本的向后兼容性。我们使用最新的稳定 API 和特性，不提供传统回退方案。

### 最佳实践

- 使用 Composition API 和 `<script setup>`
- 完整的 TypeScript 类型支持
- 遵循 WAI-ARIA 无障碍标准
- 现代 CSS 特性（CSS 变量、Grid、Flexbox）
- ES2020+ 语法特性

## 🔧 常见用法模式

### 响应式属性

```vue
<script setup lang="ts">
import { ref } from 'vue';
import { ToolTip } from '@xbghc/vue-ui';

const tooltipContent = ref('动态内容');
const isDisabled = ref(false);
</script>

<template>
  <ToolTip :content="tooltipContent" :disabled="isDisabled">
    <template #trigger>
      <button>动态 Tooltip</button>
    </template>
  </ToolTip>
</template>
```

### 事件处理

```vue
<script setup lang="ts">
import { ToolTip } from '@xbghc/vue-ui';

const handleShow = () => {
  console.log('Tooltip 显示了');
};

const handleHide = () => {
  console.log('Tooltip 隐藏了');
};
</script>

<template>
  <ToolTip content="事件演示" @show="handleShow" @hide="handleHide">
    <template #trigger>
      <button>事件演示</button>
    </template>
  </ToolTip>
</template>
```

### 富文本内容

```vue
<template>
  <ToolTip>
    <template #trigger>
      <button>富文本内容</button>
    </template>
    <div style="padding: 4px;">
      <strong>富文本 Tooltip</strong><br />
      <span style="color: #ccc;">支持任意 HTML</span>
    </div>
  </ToolTip>
</template>
```
