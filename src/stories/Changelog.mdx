# 📋 更新日志

本页面记录了 @xbghc/vue-ui 组件库的所有重要变更。

## [未发布]

### 📖 文档

- **修复**: 更新仓库URLs从 `my-component-library` 到 `vue-ui`
- **新增**: 完整的 Storybook 文档系统
- **新增**: ToolTip 组件 API 文档和样式定制说明
- **新增**: VSCode 开发配置文件

### 🔧 项目配置

- **修复**: 更新 package.json 中的仓库链接
- **新增**: .gitignore 规则支持现代开发工具
- **新增**: AI 工具配置目录支持（.claude/、.cursor/等）
- **新增**: TypeScript、Vite、ESLint 配置

### 🧹 代码重构

- **移除**: focus trigger 功能，专注 hover 触发
- **移除**: 不必要的属性（appendToBody、overlayClass、zIndex）
- **新增**: 完整的 TypeScript 类型导出
- **改进**: SCSS 样式结构
- **改进**: 导出结构，支持 Tree Shaking

### ✨ 初始功能开发

- **ToolTip组件**: 工具提示组件
  - 支持hover触发方式
  - 支持12种位置布局（top、bottom、left、right及其变体）
  - 支持自定义偏移量和延迟时间
  - 支持箭头显示/隐藏
  - 支持禁用状态
  - 支持富文本内容（通过插槽）

### 🎨 样式特性
